# NotVanilla ⚔️🧭

Um modpack de Minecraft focado em **exploração profunda**, **movimentação fluida**, **combate técnico** e **desafios intensos**. Desenvolvido para transformar a experiência vanilla em uma jornada épica e recompensadora.

## 🚀 Pilares do Projeto
- **Exploração:** Novos biomas, estruturas e segredos escondidos pelo mapa.
- **Combate & Desafio:** IA de mobs aprimorada e mecânicas de luta mais dinâmicas.
- **Movimentação:** Ferramentas e habilidades para atravessar o mundo com agilidade.
- **Performance:** Otimizado com as melhores tecnologias Fabric para 1.21.1.

## 🛠️ Desenvolvimento
Este projeto utiliza o [packwiz](https://packwiz.infra.link/) para gerenciamento de mods.

- **Workspace:** `~/workspace/minecraft/modpacks/notvanilla`
- **Produção:** `/srv/servers/minecraft/notvanilla`

---
*Criado e gerenciado no servidor Debian 12.*
